# Car rent Dashboard

## Description

Dashboard and booking view.

## Getting Started

### Dependencies

This project was made with

* reactJs 18.8.2
* typescript 4.9.3
* react-router-dom 6.6.1
* material UI 5.11.3


### Installing

* $ git clone https://github.com/IngeJob/rent-car-dashboard.git
* $ npm install
* $ npm run dev
